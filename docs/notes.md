# ملاحظات مشروع News-WebApp

هذا المشروع عبارة عن نظام ادارة اخبار مبني باستخدام PHP و MySQL
يسمح للمستخدمين بالتسجيل وتسجيل الدخول واضافة وتعديل الاخبار
تم تقسيم الاخبار حسب النوع مع امكانية رفع الصور وعرضها ضمن الواجهة

تم التركيز على بساطة التصميم وتجربة المستخدم
كما ان الواجهة متجاوبة وتعمل على مختلف احجام الشاشات

تم استخدام Docker لتسهيل تشغيل المشروع وضمان بيئة موحدة
بحيث يمكن لأي مطور تشغيل المشروع بدون الحاجة لاعدادات يدوية

## التحديات التقنية
اكبر تحدي واجهته كان اعداد الاتصال بين PHP و MySQL داخل Docker
خصوصا مع MySQL 8 الذي لا يسمح باستخدام root بدون كلمة مرور
تم حل المشكلة عن طريق انشاء مستخدم مخصص للتطبيق داخل docker-compose

## ما تعلمته
تعلمت كيفية تنظيم مشروع برمجي بشكل احترافي
وكيفية استخدام Git و GitHub بطريقة صحيحة
بالاضافة الى كتابة Dockerfile و docker-compose وتشغيل خدمات متعددة

## البونص
- Bonus A: تم استخدام docker-compose لتشغيل خدمات PHP و MySQL بسهولة
- Bonus B: تم اضافة GitHub Actions لبناء صورة Docker تلقائيا
- Bonus C: تم استخدام multi-stage Docker build لتقليل حجم الصورة
- Bonus D: تم اضافة healthcheck و endpoint بسيط للتحقق من حالة التطبيق
- Bonus E: تم اضافة Makefile لتسهيل اوامر التشغيل
  على نظام Windows اداة make غير مثبتة افتراضيا
  لكن وجود الملف يحقق الهدف التعليمي ويخدم مستخدمي Linux و macOS
- Bonus G: تم استخدام feature branch وفتح Pull Request مع وصف واضح وصور
