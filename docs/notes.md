## البونص
- Bonus A: اضفت docker-compose.yml لتسهيل تشغيل المشروع خاصة لو في خدمات متعددة
- Bonus B: جهزت GitHub Actions workflow لبناء صورة Docker وتشغيل الاختبارات اذا موجودة
- Bonus C: استخدمت multi-stage Docker build لتقليل حجم الصورة
- Bonus D: اضفت healthcheck ونقطة endpoint بسيطة لو مطلوبة
- Bonus E: اضفت Makefile بأوامر make build, make run, make stop لتسهيل التشغيل
- Bonus G: استخدمت feature branch وفتحت Pull Request للفرع الرئيسي مع وصف نظيف وصور توضيحية
