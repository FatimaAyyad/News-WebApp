#  ملاحظات مشروع News-WebApp

هذا المشروع عبارة عن نظام ادارة اخبار مبني باستخدام PHP و MySQL
يسمح للمستخدمين بالتسجيل وتسجيل الدخول واضافة وتعديل الاخبار
تم تقسيم الاخبار حسب النوع مع امكانية رفع الصور وعرضها ضمن الواجهة

تم التركيز على بساطة التصميم وتجربة المستخدم
كما ان الواجهة متجاوبة وتعمل على مختلف احجام الشاشات

تم استخدام Docker لتسهيل تشغيل المشروع وضمان بيئة موحدة
بحيث يمكن لأي مطور تشغيل المشروع مباشرة دون الحاجة لاعدادات يدوية

---

## التحديات التقنية

من اكبر التحديات التي واجهتني كانت اعداد وتشغيل التطبيق داخل Docker
خصوصا ضبط Apache و DocumentRoot بحيث يتم عرض ملفات PHP بشكل صحيح

واجهت مشكلة Not Found عند تشغيل التطبيق
وتم حلها من خلال تعديل Dockerfile وضمان وجود index.php في المسار الصحيح
بالاضافة الى ضبط اعدادات Apache بما يتوافق مع بنية المشروع

كما كان هناك تحدي في اعداد الاتصال بين PHP و MySQL داخل Docker
خصوصا مع MySQL 8 الذي لا يسمح باستخدام root بدون كلمة مرور
وتم حل ذلك عن طريق انشاء مستخدم مخصص للتطبيق داخل docker-compose

---

## ما تعلمته

تعلمت كيفية تحليل الاخطاء المتعلقة بـ Docker و Apache
وكيفية قراءة رسائل الخطأ وتشخيص سبب المشكلة الحقيقي

كما تعلمت كيفية كتابة Dockerfile بشكل صحيح
واستخدام multi-stage build وتحسين تشغيل التطبيق داخل الحاويات

بالاضافة الى ذلك اكتسبت خبرة عملية في استخدام Git و GitHub
وانشاء commits واضحة وتوثيق المشروع بطريقة احترافية

---

## النشر ومحاكاة بيئة Production

تمت محاكاة نشر التطبيق على بيئة خارجية باستخدام Cloudflare Tunnel
حيث تم تشغيل التطبيق محليا داخل Docker وجعله متاحا عبر رابط عام مؤقت

هذا الاسلوب يحاكي بيئة Production بدون الحاجة الى VPS حقيقي
وتم توثيق نجاح النشر من خلال screenshots كما هو مطلوب في الواجب
