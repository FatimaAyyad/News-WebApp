# News-WebApp

## Overview
News-WebApp is a PHP/MySQL news management system that allows users to register, add, edit, and view news items categorized by type with image uploads. The system is user-friendly, responsive, and designed to manage dynamic news content effectively.

## Features
- User registration and login
- Add, edit, and delete news items
- Categorize news by type
- Upload and display images
- Responsive and clean UI

## Tech Stack
- PHP 8.2
- MySQL 8
- Apache
- Docker & Docker Compose

## Build & Run with Docker
1. Clone the repository:
   ```bash
   git clone https://github.com/FatimaAyyad/News-WebApp.git
   cd News-WebApp
2. Start containers:
    docker-compose up -d
3. Access project:
    http://localhost:8080
4. Stop & cleanup:
    docker-compose down
5. Testing
    Create a new account or log in with an existing one

    Add, edit, and view news

    Upload images and see them in the UI
